# Feria Digital Agropecuaria

Este es un mini proyecto web que desarrollé en Visual Studio Code como práctica de diseño y estructura de páginas web.  
La página simula una feria agropecuaria digital donde se pueden explorar diferentes secciones como tipos de cultivo, galería de fotos, historias y contacto.

---

## ¿Cuál fue el objetivo?

El objetivo principal fue aplicar los conocimientos aprendidos en:

- HTML5 (estructura de la página)
- CSS3 (diseño y estilos)
- JavaScript (navegación entre vistas)

También busqué organizar bien el código y separar correctamente los archivos para que el proyecto fuera más profesional.

---

## Tecnologías utilizadas

- HTML5  
- CSS3  
- JavaScript  
- Visual Studio Code

## Funcionalidades

- Menú principal con botones interactivos.
- Cambio dinámico entre secciones (Inicio, Cultivo, Galería, Historias y Contacto).
- Diseño adaptable usando Flexbox y Grid.
- Efectos hover en botones e imágenes.
- Sección de contacto con enlaces a correo y WhatsApp.

---

## ¿Cómo ejecutarlo?

1. Descargar o clonar el repositorio.
2. Abrir la carpeta en Visual Studio Code.
3. Ejecutar el archivo `index.html` en el navegador.

Durante las pruebas del formulario encontré algunos errores, como permitir edades menores a 12 años o números de contacto que no tenían exactamente 10 dígitos. 
Para solucionarlo agregué validaciones en JavaScript que revisan los datos antes de enviarlos y muestran un mensaje si la información no cumple las condiciones.

También utilicé una estructura semántica en HTML con etiquetas como header, main y section para organizar mejor el contenido de la página. 
La jerarquía de los elementos es importante porque mejora la accesibilidad y permite que los navegadores y lectores de pantalla entiendan mejor la estructura del sitio.

Si este proyecto creciera a nivel municipal, mejoraría el diseño de la página, agregaría una base de datos para guardar las inscripciones y optimizaría la plataforma para que pueda ser utilizada por más usuarios sin problemas.
